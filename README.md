# Micro FE (WIP)

This project implements micro FE using vite, react

## Installation

Open single folder and install

```bash

pnpm install

# or

npm install

```

## Run

Every project run on single port

```bash
# host app
pnpm build & pnpm dev

# rest
pnpm build & pnpm preview
```
